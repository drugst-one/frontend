Drugs.One was developed to be an easy to use plug-in for any kind of website. It comes with a default configuration and works out of the box so that it can be integrated without specific web development knowledge. However, we wanted Drugst.One to be fully flexible in order to fit to any kind of web design and use case. Therefore, it offers a multitude of customization options to the user regarding the functionality of the tool as well as the styling. Most of these configuration options can be set via the Drugst.One playground and can simply get copy and pasted from there.
